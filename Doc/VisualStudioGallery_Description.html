General Purpose

AIT Dependency Manager helps you manager cross-solution dependencies within Visual Studio and Team Foundation Server. It is a technology-independent NuGET inhouse alternative.

Features & Benefits

This release includes the following features:

  • Free Visual Studio extension
  • Guided editing of dependencies via Visual Editor
  • Multiple providers for: Source Control, Build Result (XAML and JSON), Subversion and File share dependencies
  • MSBuild support: MSBuild Task for calling Get and Clean Dependencies during local and central build
    DOWNLOAD BUILD INTEGRATION TASKS from our website
  • Fetch all dependencies or only direct dependencies
  • Support for compressed dependencies (Compressed with ZIP or 7-Zip)
  • Support for multi site environments (similar to Windows Server DFS)
